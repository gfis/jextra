JExtra is an incrementally extensible translator generator written in Java. The package contains:
  • an LR(1) parser generator comparable to yacc or bison, with an emphasis on XML for
    • grammar input
    • parser table output
    • abstract syntax tree output
  • several parsing algorithms using the generated push-down automaton state table, or simple finite state automata,
  • several handwritten scanners for the low-level symbol recognition of commonly used programming languages like C, Java, FORTRAN etc (other compiler projects use finite state tables generated by tools like (f)lex for this purpose),

The package is based in part on the algorithms in [1], and the corresponding EXTRA system written in PL1/1 and later in FORTRAN in 1980.

[1] Fischer, Georg: Incremental LR(1) Parser Construction as an Aid to Syntactical Extensibility. Ph.D. thesis (Forschungsbericht Nr. 102), University of Dortmund, 1980
